Jishide

Jishide diucapkan Jii-se-daa diambil dari bahasa China yang berarti real time. Jishide adalah pustaka statik (static library) yang ditulis menggunakan bahasa C dan Objective-C dan dikompilasi untuk digunakan bersama (linked) dengan aplikasi. Jishide bertanggung jawab sebagai :
  1. Real time transcoding audio dari AudioBufferList ke format Ogg Vorbis atau MP3
  2. bertindak sebagai tulang punggung dalam berkomunikasi dengan streaming server (Icecast atau Shoutcast)

Baca selengkapnya di repositori saya di Github https://github.com/icescode/jishide

Kandyta

Kandyta adalah sistem terintegrasi big data yang terdiri dari
  1. Apache Hadoop
  2. Apache Spark
  3. Apache Parquet
  4. Apache Drill
  5. Microsoft Power BI
yang ditulis diatas bahasa pemrograman Java dan Python. Kandyta adalah sebuah manajemen pengaturan Big Data multi node storage yang ditujukan untuk keperluan keperluan kritikal kedinasan. Kandyta disponsori oleh TMBigData Singapura dan Shinkowa Pharmaceutical Jepang di tahun 2016 - 2019, penerapan ujicoba dilakukan di Kota Cimahi, Jawa Barat dalam bentuk kerjasama dengan pemerintah Kota Cimahi. Kandyta adalah sistem dengan kode terbuka (open source), saat ini repositori masih bersifat tertutup dengan opsi undangan kolaborasi (Github nvitation)

Ditulis sendiri menjelang malam, 29 Mei 2019.

Lentera

Lentera adalah sebuah artificial intelligence bot universal yang dirancang untuk menjadi sebuah mesin pencari informasi di dalam aplikasi pesan instan seperti WhatsApp dan Telegram, berbeda dengan bot bot lain nya, Lentera meiliki fitur
  1. Mesin pencari tanpa harus menggunakan browser
  2. Integrasi dengan database apapun sehingga memungkinkan menjadikan Lentera menjadi aplikasi apa saja
Sebagai bonus layanan Lentera dapat bertindak sebagai translator untuk :
  1. Translator dari bahasa Inggris, Jerman, Rusia, Belanda, Turki dan lainnya ke Bahasa Indonesia
  2. Translator dari bahasa Indonesia, Jerman, Rusia, Belanda, Turki dan lainnya ke Bahasa Inggris
  3. Translator dari bahasa Indonesia, Inggris, Rusia, Belanda, Turki dan lainnya ke Bahasa Jerman

Baca selengkapnya di repositori saya di Github https://github.com/icescode/lentera
